# 🚢 Titanic Survival Prediction (Machine Learning Project)

## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques.  
It demonstrates the complete data science workflow including data preprocessing, visualization, model building, and evaluation.

---

## 🎯 Objective
To analyze the Titanic dataset and build a model that predicts survival based on features like age, gender, passenger class, and fare.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Features Used
- Pclass (Passenger Class)
- Sex
- Age
- Fare
- FamilySize (Engineered Feature)

---

## ⚙️ Machine Learning Model
- Logistic Regression

---

## 📈 Results
- Model Accuracy: **~75–80%**
- The model performs well in predicting survival based on selected features.
